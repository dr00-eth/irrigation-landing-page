## About Irrigation Protocol

> Irrigation Protocol intends to improve demand for these assets through a number of different Irrigation Tools. These tools provide Beanstalk asset holders additional LIQUIDITY and REWARDS while they wait for the underlying Beanstalk asset to fully mature.

Irrigation Protocol will create rewards for Farmers through a set of decentralized instruments called Irrigation Tools. These tools will help unlock the value of illiquid assets and further the progression of credit lines without compromising price ranges.
